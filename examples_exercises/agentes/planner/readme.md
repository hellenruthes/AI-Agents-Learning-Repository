🧠 Exercise: Scrum Master Agent (Planner–Executor)
🎯 Objective

Build an agent that:

reads the backlog from the database (Postgres)
creates an analysis plan (Planner)
executes that plan (Executor)
generates recommendations as a Scrum Master
🧩 Scenario

You have a backlog table in the database containing sprint tasks.

The user asks:

"Analyze the backlog and tell me the main sprint risks and what we should do."
